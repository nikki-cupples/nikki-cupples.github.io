Margin, Border & Padding
September 5, 2024

Utilizing the CSS Box Model, we can accurately dictate the desired parameters of content within an HTML document. Each element becomes wrapped in a box, enabling developers the freedom to craft interesting webpages. Inherently, there are four key components that establish the resulting layout for viewers: Content, Border, Padding and Margin.

Through this technique, developers can manipulate each of these four aspects to seamlessly create, manipulate, and design structured content. Each layer of the box model are automatically inserted into an element, however, we can adapt their dimensions through margin, padding and border commands. In addition, each side of the box can be edited independently of one another, an attribute that facilitates complete freedom to establish unique style dimensions.

Margin

A margin dictates the surrounding space of an element, controlling only the outer spacial area. For example, the contents of this element have been set to margin: 10px 200px 10px 200px, setting the top, right, bottom and left attributes. Essentially functioning as a clock, this ensures the direction of setting each margin is easy to remember and manipulate over time.

Border

Resting inbetween the margin and padding, a border frames the content, providing added detailing or style to a webpage. More specifically, a border lines the perimeter of an element and can be personalized through a few key commands. For instance, the code for the borders surrounding this element are as follows:

border-top-style: dashed;
border-top-width: 20px;
border-top-color: #7657ff;
border-right-style: groove;
border-right-width: 30px;
border-right-color: #e4ddff;
border-left-style: double;
border-left-width: 20px;
border-left-color: #7657ff;
border-bottom-style: double;
border-bottom-width: 20px;
border-bottom-color: #e4ddff;

Padding

In contrast to the aforementioned margin, padding increases the space within an element's border around it's content. Here, you will notice that while this element is the same size as the above border box, it appears to be bigger. Utilizing this command, it is possible to create additional space between the inner contents (this text), and it's outer edges. For example, the code used to manipulate this box is padding: 10px 100px 10px 80px; with the order of application functioning as a clock (top, right, bottom, left). By classifying both the article section and the paragraph as "padding," this has resulted in these specifications applying to both the header and the paragraph padding.
